# Career Coach Platform

A comprehensive career coaching platform with AI-powered interview practice, real job market insights, and personalized career guidance.

## 🚀 Features

- **AI Interview Practice** - Mock interviews with real-time feedback
- **Career Guidance** - Real job market data and recommendations
- **Resume Analysis** - Upload and analyze resumes
- **Performance Analytics** - Track interview performance over time
- **Job Market Insights** - Real-time job postings and salary data
- **Professional Dashboard** - Comprehensive career analytics

## 🛠 Tech Stack

- **Frontend**: React 19, Bootstrap 5, Chart.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Vercel (Frontend), Render (Backend)
